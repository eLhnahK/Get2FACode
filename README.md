## Description

Get 2FA Code with API of 2fa.live

### Dependencies

* Requires .NET Framework 4.7.2 or later ☠️

* Link: [Download NET Framework](https://dotnet.microsoft.com/en-us/download/dotnet-framework) 🤘

### Installing

* Clone Git or download zip file 👌

### Executing program

* Run Get2FACode.exe 🤟
